# 飞书智能表格搭建指南：日报与周报自动化系统

## 整体架构

本方案由 **3 张数据表** + **2 个自动化脚本** 组成，全部在飞书多维表格内完成。
你只需每天填写工作记录，系统自动生成日报和周报，无需外部服务器或 API 密钥。


## 第一步：创建三张数据表

在飞书新建一个「多维表格」，命名为 **「工作管理与报告」**。

### 表1：工作记录明细（输入层）
这是你**每天填写**工作内容的表。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日期 | 日期 | 选择工作日期 |
| 星期 | 公式 | =WEEKDAY(日期)，显示为星期几 |
| 周次 | 公式 | =WEEKNUM(日期) |
| 时间段 | 单选 | 8:30-9:00, 9:00-10:00, 10:00-11:00, 11:00-12:00, 12:00-13:15, 13:30-14:00, 14:00-15:00, 15:00-16:00, 16:00-17:00, 17:00-18:00 |
| 项目名称 | 单选 | 日报总结, BI报表, 纸重计算器, 班组产值表, 工序锁定, 业务接待, 周报例会, 生产组会议, 开发文档, 其他 |
| 工作内容 | 文本 | 具体做了什么 |
| 用时(小时) | 数字 | 单位小时，0.5表示半小时 |
| 进展情况 | 单选 | 已完成, 开展中, 部分完成, 未完成, 暂停, 取消, 不处理 |
| 问题/备注 | 文本 | 遇到的问题或备注 |

建议为该表添加一个「表单视图」，方便在手机上填写。

### 表2：日报统计（自动生成）
| 字段名 | 类型 | 说明 |
|--------|------|------|
| 日期 | 日期 | 脚本自动写入 |
| 项目名称 | 文本 | 脚本自动写入 |
| 总用时(小时) | 数字 | 脚本自动计算 |
| 工作条目明细 | 文本 | 脚本汇总当日该项目的所有工作 |
| 进展状态 | 文本 | 脚本汇总各项进展状态 |
新建数据表，命名为「日报统计」。

### 表3：周报统计（自动生成）
| 字段名 | 类型 | 说明 |
|--------|------|------|
| 周起始日 | 日期 | 该周周一日期 |
| 日期范围 | 文本 | 如 6.8-6.13 |
| 项目名称 | 文本 | 脚本自动写入 |
| 周一用时至周六用时 | 数字 x6 | 脚本按天写入 |
| 周总用时(小时) | 数字 | 脚本自动计算 |
| 进展汇总 | 文本 | 脚本自动汇总 |
新建数据表，命名为「周报统计」。


## 第二步：创建自动化脚本

飞书多维表格顶部有「脚本」按钮（图标 {}），点击后新建脚本。

### 脚本1：生成日报

新建脚本命名为「生成日报」，粘贴以下代码：

`javascript
// 生成日报：读取工作记录明细 -> 按项目汇总 -> 写入日报统计
const detailTable = base.getTable("工作记录明细");
const reportTable = base.getTable("日报统计");
// 配置区：要生成日报的日期，留空则默认昨天
const TARGET_DATE = "";

async function main() {
  let targetDate;
  if (TARGET_DATE) { targetDate = new Date(TARGET_DATE); }
  else { targetDate = new Date(); targetDate.setDate(targetDate.getDate() - 1); }
  const dateStr = formatDate(targetDate);
  console.log("正在生成 " + dateStr + " 的日报...");
  const detailRecords = await detailTable.getRecords();
  const dayRecords = detailRecords.filter(r => {
    if (!r.fields["日期"]) return false;
    return formatDate(new Date(r.fields["日期"])) === dateStr;
  });
  if (dayRecords.length === 0) { console.log("该日期没有工作记录"); return; }
  const groups = {};
  dayRecords.forEach(r => {
    const project = r.fields["项目名称"] || "(未分类)";
    if (!groups[project]) groups[project] = { totalHours: 0, items: [], statuses: [] };
    groups[project].totalHours += Number(r.fields["用时(小时)"] || 0);
    if (r.fields["工作内容"]) groups[project].items.push(r.fields["工作内容"]);
    if (r.fields["进展情况"]) groups[project].statuses.push(r.fields["进展情况"]);
  });
  const oldReports = await reportTable.getRecords();
  const toDelete = oldReports.filter(r => {
    if (!r.fields["日期"]) return false;
    return formatDate(new Date(r.fields["日期"])) === dateStr;
  });
  if (toDelete.length > 0) await reportTable.deleteRecords(toDelete.map(r => r.id));
  const newRecords = Object.entries(groups).map(([project, data]) => ({
    fields: {
      "日期": targetDate.getTime(),
      "项目名称": project,
      "总用时(小时)": Math.round(data.totalHours * 100) / 100,
      "工作条目明细": data.items.filter(Boolean).join("\n"),
      "进展状态": [...new Set(data.statuses)].join("、")
    }
  }));
  await reportTable.addRecords(newRecords);
  console.log("日报生成完成，共 " + newRecords.length + " 个项目");
}
function formatDate(d) {
  return d.getFullYear() + "-" + String(d.getMonth()+1).padStart(2,"0") + "-" + String(d.getDate()).padStart(2,"0");
}
await main();
`

### 脚本2：生成周报

新建脚本命名为「生成周报」，粘贴以下代码：

`javascript
// 生成周报：读取工作记录明细 -> 按项目+天汇总 -> 写入周报统计
const detailTable = base.getTable("工作记录明细");
const weeklyTable = base.getTable("周报统计");
// 配置区：周一起始日，留空则自动取上周一
const MONDAY_DATE = "";

const dayFieldNames = ["周一用时","周二用时","周三用时","周四用时","周五用时","周六用时"];

async function main() {
  let monday;
  if (MONDAY_DATE) { monday = new Date(MONDAY_DATE); }
  else { monday = getLastMonday(); }
  const saturday = new Date(monday); saturday.setDate(monday.getDate() + 5);
  const mondayStr = formatDate(monday); const saturdayStr = formatDate(saturday);
  console.log("正在生成 " + mondayStr + " 至 " + saturdayStr + " 的周报...");
  const detailRecords = await detailTable.getRecords();
  const weekRecords = detailRecords.filter(r => {
    if (!r.fields["日期"]) return false;
    const ds = formatDate(new Date(r.fields["日期"]));
    return ds >= mondayStr && ds <= saturdayStr;
  });
  if (weekRecords.length === 0) { console.log("该周没有工作记录"); return; }
  const projects = {}; const weekDays = [];
  for (let i = 0; i < 6; i++) { const d = new Date(monday); d.setDate(monday.getDate() + i); weekDays.push(formatDate(d)); }
  weekRecords.forEach(r => {
    const project = r.fields["项目名称"] || "(未分类)";
    const dateStr = formatDate(new Date(r.fields["日期"]));
    const hours = Number(r.fields["用时(小时)"] || 0);
    const status = r.fields["进展情况"] || "";
    if (!projects[project]) projects[project] = {};
    if (!projects[project][dateStr]) projects[project][dateStr] = { hours:0, statuses:[] };
    projects[project][dateStr].hours += hours;
    if (status) projects[project][dateStr].statuses.push(status);
  });
  const oldWeeklies = await weeklyTable.getRecords();
  const toDelete = oldWeeklies.filter(r => {
    if (!r.fields["周起始日"]) return false;
    return formatDate(new Date(r.fields["周起始日"])) === mondayStr;
  });
  if (toDelete.length > 0) await weeklyTable.deleteRecords(toDelete.map(r => r.id));
  const newRecords = Object.entries(projects).map(([project, dayData]) => {
    const record = { fields: {
      "周起始日": monday.getTime(),
      "日期范围": (monday.getMonth()+1)+"."+monday.getDate()+"-"+(saturday.getMonth()+1)+"."+saturday.getDate(),
      "项目名称": project,
      "周总用时(小时)": 0
    }};
    let totalHours = 0; const allStatuses = new Set();
    for (let i = 0; i < 6; i++) {
      const data = dayData[weekDays[i]];
      const dayHours = data ? Math.round(data.hours * 100) / 100 : 0;
      record.fields[dayFieldNames[i]] = dayHours; totalHours += dayHours;
      if (data) data.statuses.forEach(s => allStatuses.add(s));
    }
    record.fields["周总用时(小时)"] = Math.round(totalHours * 100) / 100;
    record.fields["进展汇总"] = [...allStatuses].join("、");
    return record;
  });
  newRecords.sort((a,b) => b.fields["周总用时(小时)"] - a.fields["周总用时(小时)"]);
  await weeklyTable.addRecords(newRecords);
  console.log("周报生成完成，共 " + newRecords.length + " 个项目");
  newRecords.forEach(r => console.log("  " + r.fields["项目名称"] + ": " + r.fields["周总用时(小时)"] + "h"));
}
function formatDate(d) {
  return d.getFullYear()+"-"+String(d.getMonth()+1).padStart(2,"0")+"-"+String(d.getDate()).padStart(2,"0");
}
function getLastMonday() {
  const now = new Date(); const dow = now.getDay();
  const daysSince = dow === 0 ? 6 : dow - 1;
  const lastMonday = new Date(now);
  lastMonday.setDate(now.getDate() - daysSince - 7); return lastMonday;
}
await main();
`


## 第三步：设置定时自动化

1. 在多维表格顶部点击「自动化」
2. 新建自动化规则：
   - 触发器：定时触发 - 每周 - 周一 09:30
   - 动作：运行脚本 - 选择「生成周报」
3. 保存并启用

日报建议手动触发：每天下班前填写完工作后，运行一次「生成日报」脚本即可（1~2 秒完成）。


## 使用流程

每天：
1. 在工作记录明细表中填写当天各时段的工作
2. 下班前运行「生成日报」脚本
3. 打开「日报统计」表查看当日汇总

每周：
1. 周一早上打开飞书，查看「周报统计」表
2. 上周周报已自动生成（如未自动运行，手动点一次「生成周报」）
3. 直接用于 10:00 的组内工作汇报


## 扩展建议

仪表盘：多维表格支持仪表盘，可添加柱状图/饼图/折线图展示工作用时。
项目清单：创建一张「项目清单」表，作为项目名称选项的数据源。
周报导出：在周报统计表创建「看板视图」，按周筛选截图即可用于周会。


## 常见问题

Q: 脚本报错「base is not defined」？
A: 确保脚本在飞书多维表格的脚本编辑器中运行，飞书会自动注入 base 对象。

Q: 字段名不一致导致脚本出错？
A: 将脚本中 r.fields["项目名称"] 等改为你实际设置的字段名。

Q: 要生成过去某天的日报？
A: 修改 TARGET_DATE 变量，如 const TARGET_DATE = "2026-06-10";

Q: 要生成特定周的周报？
A: 修改 MONDAY_DATE 变量，如 const MONDAY_DATE = "2026-06-01";
