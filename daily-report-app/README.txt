========================================
  日报与周报管理工具 v1.2
  安装与使用说明
========================================

【简介】
一款纯本地的日报与周报管理工具。按时间段记录每日工作，自动
统计日报和生成周报。支持通过 Server酱 推送日报到微信。

【所需环境】
1. Node.js (版本 18 或更高)
   下载: https://nodejs.org/

【安装步骤】
1. 将 daily-report-app 文件夹复制到任意位置
   例如: D:\MyTools\daily-report-app

2. 创建数据存储目录（用于存放日报数据）
   在 D 盘根目录创建 ReportData 文件夹:
   md D:\ReportData

3. 启动服务器
   双击 start.bat，或在终端中执行:
   cd daily-report-app
   node server.js

4. 打开浏览器访问:
   http://localhost:3456

========================================
  使用说明
========================================

【日报填写】
- 选择日期，填写工作记录
- 时间段: 选择开始和结束时间，用时自动计算
- 工作分类: 可下拉选择，或输入新分类后点击 + 号添加
- 点击已有记录可快速填充到表单进行修改

【日报统计】
- 选择日期，自动汇总当日各分类工作用时

【周报统计】
- 默认显示上周数据，也可选择本周或其他周次
- 按工作分类展示周一至周六的每天用时和总计

========================================
  文件结构
========================================

daily-report-app/
  server.js      - 后端服务器 (零依赖，Node.js 内置模块)
  start.bat      - 一键启动脚本
  package.json   - 项目元信息
  public/
    index.html   - 主页面
    css/style.css  - macOS 风格样式
    js/app.js      - 核心交互逻辑
    js/settings.js - 推送配置逻辑

【数据存储】
D:\ReportData\
  YYYY-MM-DD.json  - 每天的工作记录
  _categories.json - 工作分类列表
  _push_config.json - 推送配置

【注意事项】
- 本工具完全本地运行，数据不经过任何外部服务器
- 服务器端口为 3456，如需修改可在 server.js 顶部修改 PORT
- 首次使用请先创建 D:\ReportData 目录
- 如果 3456 端口被占用，修改 PORT 后重启即可
